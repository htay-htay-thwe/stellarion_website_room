# Assets Directory

This directory contains app assets including icons and splash screens.

## Required Assets for Production

### App Icon
- **File**: `icon.png`
- **Size**: 1024x1024 px
- **Format**: PNG with transparency
- **Usage**: App launcher icon

### Splash Screen
- **File**: `splash.png`
- **Size**: 1242x2436 px (iPhone X resolution)
- **Format**: PNG
- **Usage**: App loading screen

### Adaptive Icon (Android)
- **File**: `adaptive-icon.png`
- **Size**: 1024x1024 px
- **Format**: PNG with transparency
- **Usage**: Android adaptive icon

### Favicon (Web)
- **File**: `favicon.png`
- **Size**: 48x48 px or larger
- **Format**: PNG
- **Usage**: Web version favicon

## Creating Icons

You can use the following tools to create icons:

1. **Figma** - https://www.figma.com/
2. **Canva** - https://www.canva.com/
3. **Icon Generator** - https://www.appicon.co/

## Default Icons for Development

For now, Expo will use default icons. Replace these files before publishing to App Store/Play Store.

## Recommended Design

### Icon Design Ideas:
- 3D cube with AR elements
- Furniture silhouette with AR overlay
- Camera with 3D model emerging
- Modern, minimalist design
- Brand colors: #007AFF (blue) and white

### Splash Screen Design:
- App logo centered
- Background: White or gradient
- Loading indicator
- Tagline: "Visualize Your Space"
